# Atividade 1 - Criação de Página HTML

## Desenvolva uma página HTML com tema livre (ex: hobby, tecnologia, viagem, jogos, etc.). A página deve apresentar uma estrutura organizada, semântica e legível.

 1. Estrutura básica
    Utilize corretamente as seguintes tags:
    <html>, <head>, <body>
    Inclua os metadados comuns (título na aba do navegador, ícone, codificação, idioma)

 2. Estrutura semântica
    A página deve conter as tags que garantem semântica e acessibilidade.


 3. Conteúdo
    Um título principal 
    Pelo menos dois subtítulos
    Parágrafos explicando o tema

 4. Listas
    Uma lista ordenada
    Uma lista não ordenada
    Mínimo de 3 itens em cada

 5. Links
    Um link externo (abrindo em nova aba)
    Um link interno (âncora na própria página)

 6. Imagem
    Inserir pelo menos uma imagem com:
    src
    alt 

 7. Organização do código
    Código indentado
    Uso de comentários explicativos (1 comentário obrigatório)

    Extra (Opcional):
    iframe
    canvas
    emojis

 Observações
    O tema deve ter conteúdo próprio (evite páginas muito genéricas);
    A organização e clareza serão consideradas na avaliação;
    Não utilize CSS e JavaScript, o foco da atividade é o HTML;
    O objetivo é o aprendizado. Cópias ou atividades idênticas serão zeradas.
    Formato de Entrega e Prazo
    Envie o link do repositório do GitHub (acesso público);
    Atenção ao prazo, a atividade não será reaberta.
